## 🛠️ How to contribute

You can contribute in two ways:

### 👉 Option 1: Submit an Idea (Recommended)
- Go to **Issues → New Idea**
- Fill out the structured template  
- Submit  

### 👉 Option 2: Add via PR
1. Fork this repo  
2. Add your idea inside `/IDEAS/`  
3. Follow the format below  
4. Create a Pull Request  

---
